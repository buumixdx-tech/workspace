# 【深度研报】002243 力合科创：脑机接口概念关联分析报告

**分析对象：** **002243**（股票名称：**力合科创**，全称：深圳市力合科创股份有限公司）
**归类原因：** 其被归类为“脑机接口概念”主要源于其在 **科技孵化**、**股权投资** 以及 **产学研转化** 三个方面的布局。

| 评分项 | 评估结果 |
| --- | --- |
| **综合评分** | ⭐⭐⭐ (3星) |
| **评级标签** | 参股/孵化 / 影子股 / 早期研发 |
| **核心理由** | 公司自身不生产脑机接口设备，核心逻辑是“挖到了金矿”。其投资孵化的**微灵医疗**是国内植入式脑机接口的领军企业。属于典型的“创投型”概念股。 |

**定性总结：** 与其说它是一家“脑机接口制造商”，不如说它是一只持仓了顶级脑机接口初创企业的**“科技创投基金”**。

### 一、 核心原因：为什么它是脑机接口概念？

002243 属于产业链的 **外围投资方**。市场将其纳入该板块主要基于以下实锤信息（源自公告、研报或互动易回复）：

#### 1. 参股微灵医疗（最硬核的实锤）

这是公司最实质性的相关业务关联。微灵医疗（Weilan Medical）是国内极少数对标马斯克 Neuralink 的企业。

* **具体布局：** 力合科创通过旗下的投资平台（如珠海力合华金等）参与了**微灵医疗**的早期投资。根据互动易回复：“公司投资孵化企业微灵医疗主要从事医疗级全植入式脑机接口系统研发”。
* **技术/业务原理：** 微灵医疗专注于**高通量植入式脑机接口**，技术路线是MEMS电极阵列，旨在通过植入大脑皮层的传感器采集神经信号，帮助瘫痪患者精准控制外部设备。这是脑机接口皇冠上的明珠，技术难度远高于非侵入式（头戴帽）。

#### 2. 背靠深圳清华大学研究院（产学研底座）

力合科创的母体是深圳清华大学研究院的控股平台。

* **合作背景：** 清华大学在神经工程和脑机接口领域处于国内顶尖水平。力合科创作为转化平台，天然拥有接触并孵化清华系脑机接口项目的优先权。
* **研发/应用方向：** 公司致力于将高校的“硬科技”成果转化为商业项目，脑科学是其重点关注的未来产业赛道之一。

### 二、 标的深度挖掘：业务与基本面

要判断这只股票的投资价值，不能只看概念，必须看其核心业务底座。

#### 1. 核心业务：科技创新服务 + 新材料

力合科创的本质是一家 **“科创服务+产业园区”运营商**，同时保留了原有的化妆品包装材料业务。

* **旗舰资产：** 分布在全国的**力合科创园**（孵化器）以及持有的众多硬科技初创企业股权。
* **商业模式：** “房东+股东”模式。一方面收园区租金和服务费，另一方面通过投资初创企业，通过上市或并购退出获得投资收益。

#### 2. 脑机接口对业绩的贡献

* **现状：** **无直接营收贡献**。微灵医疗尚处于研发和临床试验阶段，并未产生大规模利润，且力合科创仅为参股股东，无法并表营收。
* **投资逻辑错位：** 投资者买它通常是看中其作为**“脑机接口影子股”**的期权价值。当微灵医疗有重大融资或技术突破（如人体临床试验成功）时，力合科创的估值会因“持股升值预期”而提升。

#### 3. 财务状况与风险

* **当前阶段：** 业绩受宏观经济和创投环境影响较大。投资收益波动性强。
* **核心看点：** 市场关注其孵化企业的IPO进程（“金蛋”什么时候孵出来）。

### 四、 总结与建议

**002243 力合科创 属于“创投型”脑机接口概念股。**

* **原因：** 它不是自己做脑机接口，它是**投了做脑机接口的人**。它拥有国内顶尖的孵化网络，捕捉到了微灵医疗这个行业独角兽。
* **定性：** 它的主业是**科创服务**，脑机接口是其投资版图中的一颗**明星种子**。
**下一步行动建议：**
* **短线题材博弈：** 当市场炒作“植入式脑机接口”且提到“微灵医疗”时，力合科创是唯一的A股映射标的，具有稀缺性，可作为补涨关注。
* **中长线价值投资：** 逻辑在于看好中国硬科技创投的未来。单纯为了脑机接口概念长持该股的效率可能不如直接持有相关ETF或纯正医疗股。
