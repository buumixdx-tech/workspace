## 1. 主数据录入 (`entry.py`)

### 渠道客户
| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `name` | 文本 | ✅ | 客户名称 |
| `info` | 文本 | ❌ | 整体信息描述 |

### 点位
| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `owner_type` + `owner_id` | 选择 | ✅ | 归属主体：闪饮自身/客户/供应商 |
| `name` | 文本 | ✅ | 点位名称 |
| `type` | 选择 | ✅ | 类型 |
| `address` | 文本 | ✅ | 详细地址 |
| `receiving_address` | 文本 | ❌ | 收货地址，默认同详细地址 |

### 供应商
| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `name` | 文本 | ✅ | 供应商名称 |
| `category` | 选择 | ✅ | 供应类别 |
| `address` | 文本 | ❌ | 地址信息 |

### SKU
| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `supplier_id` | 选择 | ✅ | 所属供应商 |
| `name` | 文本 | ✅ | SKU名称 |
| `type_level1` | 选择 | ✅ | 一级分类（设备/物料） |
| `model` | 文本 | ❌ | 型号 |

### 外部合作方
| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `name` | 文本 | ✅ | 名称 |
| `type` | 选择 | ✅ | 类型（运输/客服/维修/其他） |

### 银行账户
| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `owner_type` + `owner_id` | 选择 | ✅ | 归属方：闪饮自身/客户/供应商/合作方 |
| `account_info.holderName` | 文本 | ✅ | 开户名称 |
| `account_info.bankName` | 文本 | ✅ | 银行名称 |
| `account_info.accountNo` | 文本 | ✅ | 银行账号 |
| `is_default` | 布尔 | ✅ | 设为默认账户 |

| `contract_number` | 文本 | ❌ | 合同编号 |
| `contract_files` | 文件上传 | ❌ | 供货合同附件 |
